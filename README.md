# selenium_demo

Selenium Web Driverの使い方のデモを行う為のレポジトリです

# Requirement

- ruby
 - rubygems
 - bundler

# Setup

```
# clone code
$ git clone https://github.com/takaidohigasi/selenium_demo.git
$ cd selenium_demo

# install libraries
$ bundle install --path=vendor/bundle
```

# Sample

```
$ bundle exec ruby bin/ruby_warrior_demo.rb
```
